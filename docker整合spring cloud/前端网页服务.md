# 前端服务搭建

## 1. 安裝node

## 2. 安装Vue cli

利用node package manager：npm 来安装vue-cli

配置npm镜像

```bash
npm config set registry https://registry.npm.taobao.org
```

输入命令:

```bash
npm install -g @vue/cli
```

### 3. 创建项目

通过vue命令

```bash
vue create frontapp
```

选择Manaully select features

选择 babel vuex router （用空格键）

## 4. 安装axios插件

```bash
 vue add axios
```

## 5. 运行项目

进入项目根目录

```bash
cd frontapp
npm run serve
```
